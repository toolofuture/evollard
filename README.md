**Quantum Art Authentication Service**

## Local Development

Simply open `index.html` in your browser. No build process required.

```bash

git clone https://github.com/toolofuture/evollard.git

# Navigate to directory
cd evollard

# Open in browser
start index.html  # Windows
open index.html   # macOS
```

## Design Philosophy

**"Vollard's Voice"**: The application embodies a persona that is both a classical art connoisseur and a quantum computing expert. Every UI element, from typography to color choice, reinforces this duality:

- **Gold (#d4af37)**: Heritage, value, artistry
- **Cyan (#00f3ff)**: Quantum, technology, precision
- **Dark Mode**: Gallery ambiance

## Simulation Details

This is a **demonstration application**. The following features are simulated:

- Art authentication analysis
- Payment processing
- Transaction IDs

No real AI, quantum computing, or payment processing is involved.
